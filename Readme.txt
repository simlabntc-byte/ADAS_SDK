1) Installare la versione di Python

	1.1) Direttamente dal sito Python. Per la seguente cartella --> 3.11


2) Creare il virtual environment dentro la cartella del Progetto

	2.1) cd /path/del/progetto\File SDK
	2.2) python3.11 -m venv .venv

3) Attivare il virtual environment
	
	3.1) .venv\Scripts\activate

4) Installare tutti i requirements

	4.1) pip install -r File SDK/requirements.txt
